## **3. Screenshot & Screen Recording Control**

Aerlot includes a **privacy-focused anti-capture system**:

- If **either user** has `screenCapture: true` ‚Üí screenshots and screen recording are **blocked** on the ChatRoom screen.  
- If **both users** have `screenCapture: false` ‚Üí screenshots and screen recording are **allowed**.

This preference is stored in your profile and can be changed at any time.
